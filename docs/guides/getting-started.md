## 5-Minute Quick Start (for developers)

If you already have Python and Git installed:

!!! tip "30-Second Overview"
    **Berta Chapters** is a free, open-source AI curriculum where every chapter has executable code, notebooks, and exercises -- all on GitHub, generated by [Berta AI](https://berta.one), curated by Luigi Pascal Rondanini.

**Step 1: Clone and install**

```bash
git clone https://github.com/luigipascal/berta-chapters.git
cd berta-chapters
pip install -r requirements.txt
```

**Step 2: Launch the Interactive Hub**

```bash
python interactive/berta.py
```

The hub offers skill assessment, learning paths, progress tracking, and knowledge quizzes.

**Step 3: Open your first notebook**

```bash
cd chapters/chapter-01-python-fundamentals
jupyter notebook notebooks/01_introduction.ipynb
```

That's it—you're learning!
