# Chapter 6: Introduction to Machine Learning

**Track**: Practitioner | **Time**: 8 hours | **Prerequisites**: Chapters 1-4

---

## Learning Objectives

By the end of this chapter, you will be able to:

- Differentiate supervised, unsupervised, and reinforcement learning
- Build your first ML model: linear regression from scratch
- Implement train/validation/test splits and cross-validation
- Apply feature engineering to transform raw data
- Evaluate models with accuracy, precision, recall, F1, and confusion matrices
- Understand the bias-variance tradeoff and detect overfitting/underfitting
- Use regularization to improve model generalization
- Complete an end-to-end ML project: customer churn prediction
- Compare multiple models and select the best using proper evaluation
