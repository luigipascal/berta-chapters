# Chapter 4: Probability & Statistics for Machine Learning

**Track**: Foundation | **Time**: 8 hours | **Prerequisites**: Chapter 1, basic algebra

---

## Learning Objectives

By the end of this chapter, you will be able to:

- Understand probability fundamentals: sample spaces, events, conditional probability
- Work with key distributions: Bernoulli, Binomial, Normal, Poisson
- Apply Bayes' theorem to spam classification and medical diagnosis
- Simulate and visualize the Law of Large Numbers and Central Limit Theorem
- Conduct hypothesis testing (z-test, t-test) for ML experiments
- Design and analyze A/B tests for model comparison
- Compute confidence intervals and bootstrap estimates
- Interpret p-values and avoid common statistical misconceptions
