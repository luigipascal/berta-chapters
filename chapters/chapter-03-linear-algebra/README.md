# Chapter 3: Linear Algebra & Calculus for Machine Learning

**Track**: Foundation | **Time**: 10 hours | **Prerequisites**: Chapter 1

---

## Learning Objectives

By the end of this chapter, you will be able to:

- Understand and implement vector operations (addition, scalar multiply, dot product)
- Compute vector norms (L1, L2) and apply them in similarity metrics
- Work with matrices and tensor operations (multiply, transpose)
- Grasp linear transformations as matrix multiplication
- Compute derivatives and partial derivatives
- Implement gradient descent and understand the chain rule
- Build linear regression from scratch using gradient descent
