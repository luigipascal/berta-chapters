# NLP Chapter 10 Datasets

Educational datasets for **Chapter 10: Natural Language Processing Basics**. Use them for sentiment analysis, text classification, and named entity recognition practice.

---

## reviews.csv

Sentiment-labeled reviews for binary classification.

- **Columns:** `text`, `sentiment`, `rating`
- **sentiment:** 0 = negative, 1 = positive
- **rating:** 1–5 stars (optional)
- **Size:** 15 examples (mix of movie, product, food, app, hotel, and general reviews)

**Use cases:**
- Sentiment analysis projects
- Text classification baseline
- TF-IDF + logistic regression training
- Model evaluation and comparison

---

## news_categories.csv

Short news-style excerpts with topic categories.

- **Columns:** `text`, `category`
- **Categories:** Technology, Politics, Sports, Entertainment, Science
- **Size:** 20 examples (4 per category)

**Use cases:**
- Multi-class text classification
- Category prediction
- Document clustering and similarity
- Feature importance and interpretability

---

## ner_examples.txt

Plain-text sentences suitable for named entity recognition. Not formally annotated; use with spaCy or other NER models to extract PERSON, ORG, GPE, DATE, etc.

- **Size:** 10 diverse sentences
- **Entity types:** Persons, organizations, locations, dates

**Use cases:**
- NER pipeline practice
- Evaluating pre-trained NER models
- Error analysis and entity boundary discussion
