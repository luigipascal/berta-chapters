# Chapter 7: Supervised Learning - Regression & Classification

**Track**: Practitioner | **Time**: 10 hours | **Prerequisites**: Chapters 1-6

---

## Learning Objectives

By the end of this chapter, you will be able to:

- Implement linear and polynomial regression from scratch using NumPy
- Understand regularization (Ridge L2, Lasso L1) and coefficient shrinkage
- Build logistic regression from scratch for binary classification
- Visualize decision boundaries for classification models
- Implement decision trees and understand splitting criteria
- Apply Support Vector Machines with the kernel trick
- Evaluate models with ROC curves, AUC, and precision-recall curves
- Handle imbalanced classes with oversampling and class weights
- Implement ensemble methods: bagging from scratch, Random Forest, Gradient Boosting
- Perform hyperparameter tuning with GridSearchCV and RandomizedSearchCV
- Build a complete credit risk classification pipeline end-to-end
